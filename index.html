BOLA · Broken Object Level Authorization

Autenticar ≠ Autorizar

✅ Requisição legítima

GET /api/orders/1001

→ Pedido do próprio usuário

❌ ID manipulado

GET /api/orders/1002

→ Dados de outro usuário retornados

Antiga. Conhecida. Devastadora. Testaram sucesso, não isolamento.
